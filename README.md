# [Pyramid](https://trypyramid.com/)

Framework Web para construção de aplicações web, com a ideia de ser simples e rápido de começar, com o menos de código possível, mas podendo expandir e crescer para algo grande.

## WSGI (Web Server Gateway Interface)]
É uma *calling convention* para servidores web encaminhares requisições à aplicações web ou frameworks escritas em Python. Ou seja, Python oferece uma interface agnóstica de implementação entre aplicações web e servidores web para promover um terreno comum para desenvolvimento de aplicações Web **portáveis**.
